🎨 EFFETS VISUELS DÉTAILLÉS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PARALLAX HORIZONTAL
───────────────────

1. APPARITION INITIALE
   → Les cartes arrivent depuis les côtés :
      - Cartes impaires : depuis la GAUCHE (translateX -100px)
      - Cartes paires : depuis la DROITE (translateX +100px)
   → Fade in progressif (opacity 0 → 1)
   → Délai de 150ms entre chaque carte (effet cascade)

2. DÉFILEMENT CONTINU
   → Mouvement lent pendant le scroll :
      - Cartes de gauche : se déplacent légèrement vers la gauche
      - Cartes de droite : se déplacent légèrement vers la droite
   → Speed : 0.3 (très lent et fluide)
   → Utilisation de requestAnimationFrame pour optimisation

3. HOVER EFFECTS
   → Levée de la carte (translateY -10px)
   → Scale léger (1.02)
   → Changement de bordure (blanc plus opaque)
   → Effet de brillance qui traverse la carte (shine effect)
   → Icône qui scale et rotate légèrement


DESIGN DES CARTES
─────────────────

→ Background : Semi-transparent noir (0.03 opacity)
→ Border : Bordure fine blanche semi-transparente
→ Border-radius : 20px (coins arrondis)
→ Padding : 3rem 2rem
→ Grid : 3 colonnes desktop, 1 colonne mobile


ICÔNES ANIMÉES
──────────────

→ Taille : 4rem (grandes et visibles)
→ Animation au hover : scale(1.2) + rotate(5deg)
→ Transition fluide 0.3s


TYPOGRAPHIE
───────────

→ Titre : Bebas Neue (UPPERCASE, letterspacing 0.05em)
→ Texte : Inter (rgba blanc 0.7 opacity, line-height 1.6)

💻 CODE TECHNIQUE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

HTML (index.html)
─────────────────

<section class="section-pourquoi">
    <div class="raisons-grid">
        <div class="raison-card scroll-horizontal-left">
            → Carte qui arrive de la GAUCHE
        </div>
        <div class="raison-card scroll-horizontal-right">
            → Carte qui arrive de la DROITE
        </div>
        ...
    </div>
</section>


CSS (style.css)
───────────────

.scroll-horizontal-left {
    opacity: 0;
    transform: translateX(-100px);  → Départ à gauche
}

.scroll-horizontal-right {
    opacity: 0;
    transform: translateX(100px);   → Départ à droite
}

.is-visible {
    opacity: 1;
    transform: translateX(0);        → Position finale
}


JAVASCRIPT (script.js)
──────────────────────

1. IntersectionObserver
   → Détecte quand les cartes entrent dans le viewport
   → Ajoute la classe "is-visible" avec délai progressif
   → Threshold : 0.2 (20% visible)
   → RootMargin : -100px en bas (déclenche avant)

2. Scroll Listener avec requestAnimationFrame
   → Effet de parallax continu pendant le scroll
   → Calcul de la position basé sur scrolled * speed
   → Speed : 0.3 (très lent)
   → Optimisé avec "ticking" flag (pas de lag)
