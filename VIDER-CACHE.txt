✅ SOLUTIONS (choisissez-en UNE)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣ RECHARGEMENT FORCÉ (LE PLUS SIMPLE)
   ────────────────────────────────────

   Sur Mac :
   → Cmd + Shift + R
   
   Sur Windows/Linux :
   → Ctrl + Shift + R
   → ou Ctrl + F5

   → Cela force le navigateur à ignorer le cache


2️⃣ MODE NAVIGATION PRIVÉE
   ───────────────────────

   → Ouvrir une fenêtre de navigation privée
   → Aller sur http://localhost:8000
   → La navigation privée n'utilise pas le cache


3️⃣ VIDER COMPLÈTEMENT LE CACHE
   ────────────────────────────

   CHROME / EDGE :
   1. Ouvrir les Outils de développement (F12)
   2. Clic droit sur le bouton de rafraîchissement
   3. Choisir "Vider le cache et actualiser"

   SAFARI :
   1. Cmd + Option + E (vider le cache)
   2. Cmd + R (rafraîchir)

   FIREFOX :
   1. Cmd/Ctrl + Shift + Delete
   2. Cocher "Cache"
   3. Cliquer "Effacer maintenant"


4️⃣ OUTILS DE DÉVELOPPEMENT
   ─────────────────────────

   1. Ouvrir les DevTools (F12)
   2. Onglet "Network" ou "Réseau"
   3. Cocher "Disable cache" ou "Désactiver le cache"
   4. Garder les DevTools ouverts
   5. Rafraîchir la page (F5)
