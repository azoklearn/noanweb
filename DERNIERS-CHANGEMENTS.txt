🎨 DÉFILEMENT HORIZONTAL DÉTAILLÉ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

COMPORTEMENT
────────────

→ Section "POURQUOI AVOIR UN SITE WEB ?"
→ 6 cartes alignées horizontalement
→ Scroll vertical = défilement horizontal
→ Speed : 0.5 (moyen, fluide)
→ Scrollbar invisible


TECHNIQUE
─────────

CSS :
→ display: flex (ligne horizontale)
→ overflow-x: auto (scroll horizontal)
→ scrollbar-width: none (masquer scrollbar)
→ min-width: 350px par carte (280px mobile)
→ flex-shrink: 0 (les cartes gardent leur taille)

JavaScript :
→ Détection du scroll vertical (window.pageYOffset)
→ Conversion en scroll horizontal (scrollLeft)
→ requestAnimationFrame (performance 60fps)
→ scrollSpeed: 0.5 (ajustable)


FORMULE
───────

scrollLeft = scrolled * scrollSpeed

Où :
- scrollLeft = position horizontale de la grille
- scrolled = position verticale du scroll de la page
- scrollSpeed = vitesse de conversion (0.5 = moyen)


PERSONNALISATION
────────────────

Dans script.js, ligne ~228 :
const scrollSpeed = 0.5;

Plus rapide : 0.8 ou 1.0
Plus lent : 0.3 ou 0.2

⏳ PRELOADER OPTIMISÉ DÉTAILLÉ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CLASSES BODY
────────────

1. .preloader-active (initial)
   → Body avec overflow hidden
   → Tout le contenu caché (opacity: 0, visibility: hidden)
   → Seul le preloader est visible

2. .preloader-done (après animation)
   → Body normal
   → Contenu visible avec fade-in (opacity: 1)
   → Transition : 0.5s ease


TIMING
──────

1. Page charge → body.preloader-active
2. Écrans du preloader se cachent (animation)
3. Logo disparaît complètement
4. Attente 500ms supplémentaires
5. body.preloader-active → body.preloader-done
6. Contenu apparaît en fade-in


RÉSULTAT
────────

✅ Pas de "flash" de contenu pendant le preloader
✅ Transition fluide et professionnelle
✅ Logo disparaît avant apparition du site
✅ Expérience utilisateur premium

💻 CODE TECHNIQUE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CSS - Preloader
───────────────

body.preloader-active {
    overflow: hidden;
}

body.preloader-active > *:not(#preloader) {
    opacity: 0;
    visibility: hidden;
}

body.preloader-done > *:not(#preloader) {
    opacity: 1;
    visibility: visible;
    transition: opacity 0.5s ease;
}


CSS - Scroll horizontal
───────────────────────

.raisons-grid {
    display: flex;
    overflow-x: auto;
    scroll-behavior: smooth;
    scrollbar-width: none;
    -ms-overflow-style: none;
}

.raisons-grid::-webkit-scrollbar {
    display: none;
}

.raison-card {
    min-width: 350px;
    flex-shrink: 0;
}


JavaScript - Scroll horizontal
──────────────────────────────

const raisonsGrid = document.querySelector('.raisons-grid');

window.addEventListener('scroll', () => {
    const scrolled = window.pageYOffset;
    const scrollSpeed = 0.5;
    raisonsGrid.scrollLeft = scrolled * scrollSpeed;
});


JavaScript - Preloader
──────────────────────

// Au chargement
document.body.classList.add('preloader-active');

// Après animation complète
setTimeout(() => {
    document.body.classList.remove('preloader-active');
    document.body.classList.add('preloader-done');
}, totalAnimationTime + 500);
