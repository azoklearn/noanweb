📋 ÉTAPES À SUIVRE (DANS L'ORDRE)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ÉTAPE 1 : TEST DE LA POLICE
────────────────────────────

1. Ouvrir : http://localhost:8000/test-police.html

2. Vous devriez voir DEUX textes identiques :
   → Le premier en Space Grotesk (géométrique, moderne)
   → Le deuxième en Arial (standard)

3. Si les deux textes sont DIFFÉRENTS :
   ✅ Space Grotesk fonctionne !
   → Passez à l'étape 2

4. Si les deux textes sont IDENTIQUES :
   ❌ Problème de connexion Google Fonts
   → Vérifiez votre connexion internet
   → Redémarrez le serveur


ÉTAPE 2 : ARRÊTER LE SERVEUR
─────────────────────────────

Dans le terminal où le serveur tourne :
→ Appuyez sur Ctrl + C


ÉTAPE 3 : RELANCER LE SERVEUR
──────────────────────────────

cd /Users/marinezablot/noan-web
python3 -m http.server 8000


ÉTAPE 4 : OUVRIR EN NAVIGATION PRIVÉE
──────────────────────────────────────

1. Ouvrir une nouvelle fenêtre de navigation privée :
   
   Chrome/Edge : Cmd/Ctrl + Shift + N
   Safari : Cmd + Shift + N
   Firefox : Cmd/Ctrl + Shift + P

2. Aller sur : http://localhost:8000

3. Observer les titres :
   → "VOTRE SITE WEB SUR MESURE À NANCY"
   → "POURQUOI AVOIR UN SITE WEB ?"
   → "NOS SERVICES"

4. Si les titres ont un look géométrique/moderne :
   ✅ C'EST BON ! Space Grotesk est active !


ÉTAPE 5 : VIDER LE CACHE NORMAL
────────────────────────────────

Si ça marche en navigation privée mais pas en normal :

1. Dans votre navigateur normal
2. Appuyez sur F12 (ouvrir DevTools)
3. Clic DROIT sur le bouton rafraîchir
4. Choisir "Vider le cache et actualiser"
