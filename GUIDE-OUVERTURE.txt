🌐 MÉTHODE 1 : OUVERTURE DIRECTE (Simple mais limité)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Ouvrez le dossier : /Users/marinezablot/noan-web/
2. Double-cliquez sur le fichier : index.html
3. Le site s'ouvre dans votre navigateur par défaut

⚠️  Certaines fonctionnalités avancées peuvent ne pas fonctionner
    en mode "file://" (notamment les requêtes AJAX futures)


🔥 MÉTHODE 2 : SERVEUR LOCAL (Recommandé)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Option A - Avec Python (déjà installé sur Mac)
───────────────────────────────────────────────
1. Ouvrez le Terminal (Applications → Utilitaires → Terminal)

2. Tapez ces commandes :
   cd /Users/marinezablot/noan-web
   python3 -m http.server 8000

3. Ouvrez votre navigateur et allez sur :
   http://localhost:8000

4. Pour arrêter : Ctrl + C dans le Terminal


Option B - Avec PHP (déjà installé sur Mac)
────────────────────────────────────────────
1. Ouvrez le Terminal

2. Tapez ces commandes :
   cd /Users/marinezablot/noan-web
   php -S localhost:8000

3. Ouvrez votre navigateur et allez sur :
   http://localhost:8000

4. Pour arrêter : Ctrl + C dans le Terminal


Option C - Avec Node.js (si installé)
──────────────────────────────────────
1. Ouvrez le Terminal

2. Installez http-server (une seule fois) :
   npm install -g http-server

3. Lancez le serveur :
   cd /Users/marinezablot/noan-web
   http-server -p 8000

4. Ouvrez votre navigateur et allez sur :
   http://localhost:8000

5. Pour arrêter : Ctrl + C dans le Terminal


💻 MÉTHODE 3 : AVEC VISUAL STUDIO CODE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Téléchargez VS Code (si pas déjà installé) :
   https://code.visualstudio.com/

2. Ouvrez VS Code

3. Menu → Fichier → Ouvrir le dossier
   Sélectionnez : /Users/marinezablot/noan-web

4. Installez l'extension "Live Server" :
   - Cliquez sur l'icône Extensions (carré à gauche)
   - Cherchez "Live Server"
   - Cliquez sur "Installer"

5. Clic droit sur index.html → "Open with Live Server"

6. Le site s'ouvre automatiquement dans votre navigateur

🐛 EN CAS DE PROBLÈME
━━━━━━━━━━━━━━━━━━━━
1. Le preloader reste bloqué ?
   → Actualisez la page (Cmd + R)
   → Ouvrez la console (Cmd + Option + J) pour voir les erreurs

2. Les animations ne marchent pas ?
   → Vérifiez que JavaScript est activé
   → Testez avec un serveur local au lieu d'ouvrir directement

3. Le design est cassé ?
   → Videz le cache (Cmd + Shift + R)
   → Vérifiez que tous les fichiers sont dans le même dossier
